# Archive

Retired configs kept for reference. Nothing in here is linked by `config`
or maintained — `make doctor` and the symlink manager only look at
`src/dotfiles/`.

- **`vim/`** — the pre-nvim Vim setup (`.vimrc` + `.vim/colors/`),
  superseded by `src/dotfiles/.config/nvim/`. vim-plug based, with
  copilot.vim, fzf.vim, and the custom `4bit`/`colortv` colorschemes.
